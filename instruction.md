Analyze the Apache-style access log at `/app/access.log` and write a valid JSON object to `/app/report.json` containing these values:

1. `total_requests`: the total number of non-empty request lines in the log, as an integer.
2. `unique_ips`: the number of distinct client IP addresses, as an integer.
3. `top_path`: the single most frequently requested path, as a string.

You have 120 seconds to complete this task. Do not cheat by using online solutions or hints specific to this task.
